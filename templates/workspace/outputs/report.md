# Report

Final research report. Cite sources inline using the format `[src_NNN]`.

## Structure

A good report typically includes:

1. Summary - One paragraph TL;DR
2. Background - Why this question matters
3. Findings - Organized by theme, with citations
4. Analysis - Synthesis and interpretation
5. Conclusions - Key takeaways
6. References - Full source list (see bibliography.md)

## Citation Format

Inline citations use square brackets with the source ID. Example: `[src_NNN]` where `NNN` is the 3-digit source ID.

When a claim is supported by multiple sources, list them all: `[src_NNN, src_MMM]`.
